مرحباً بك في Vox Space

Vox Space منصة استضافة مجانية احترافية. انشر موقعك في ثوانٍ، احصل على سب-دومين تلقائي، وبنية تحتية خاصة بنا بالكامل.

الخطة المجانية تشمل 50 مشروع، سب-دومين، SSL تلقائي، ودومين مخصص — بدون بطاقة ائتمان.

ما هو Vox Space؟

منصة استضافة للمواقع الستاتيكية مع بنية تحتية خاصة بنا. تدعم HTML/CSS/JS وكل المشاريع الستاتيكية.

ما الذي تحصل عليه؟

  • 50 مشروع في الخطة المجانية
  • سب-دومين تلقائي على voxspace.qzz.io
  • SSL/TLS تلقائي على كل المشاريع
  • ربط دومين مخصص مجاناً
  • نشر عبر GitHub أو ZIP أو ملفات مباشرة
  • Analytics أساسي مدمج
  • حماية DDoS وشبكة CDN

البدء السريع

انشر أول مشروع لك في أقل من دقيقتين.

1
أنشئ حساباً

روح voxspace.qzz.io/auth/register وسجل بالإيميل أو GitHub.

2
افتح الداشبورد

من الداشبورد اضغط "مشروع جديد" واختار طريقة الرفع.

3
ارفع مشروعك

GitHub repo، ZIP، أو ملفات مباشرة — اختار الأسهل ليك.

4
موقعك live

في ثوانٍ رابطك جاهز: projectname.voxspace.qzz.io

نشر عبر GitHub

اربط GitHub repo وكل push ينشر تلقائياً.

الخطوات

1
من الداشبورد اضغط "مشروع جديد"
2
اختار "GitHub Repo"

هتطلب منك توصل GitHub account للمرة الأولى.

3
اختار الـ repo

حدد الـ branch وأضف اسم المشروع.

4
اضغط Deploy

كل push بعد كده هينشر تلقائياً.

الـ root directory يجب أن يحتوي على index.html أو يكون محدداً في الإعدادات.

رفع ملف ZIP

اضغط مشروعك في ZIP وارفعه بالسحب والإفلات.

المتطلبات

  • حجم أقصى: 100MB للملف الواحد
  • الملف الرئيسي index.html في الـ root
  • صيغة ZIP فقط (ليس RAR أو 7z)

كيف تضغط مشروعك

Terminal
# ادخل مجلد مشروعك
cd my-project/

# اضغطه في ZIP
zip -r my-project.zip .

# أو على Windows: انقر بالزر الأيمن → Send to → Compressed folder

رفع ملفات مباشرة

اسحب ملفاتك أو مجلدك كاملاً وأفلته في المنصة.

الطريقة

1
اضغط "مشروع جديد" ← "رفع ملفات"
2
اسحب المجلد أو الملفات للمنطقة المحددة
3
أضف اسم المشروع واضغط Deploy
يمكن رفع حتى 500 ملف في كل مشروع. الحجم الأقصى الإجمالي 200MB.

سب-دومين تلقائي

كل مشروع يحصل تلقائياً على رابط خاص على نطاق voxspace.

شكل الرابط

Format
projectname.voxspace.qzz.io

قواعد اسم المشروع

  • أحرف صغيرة وأرقام وشرطة (-) فقط
  • لا يبدأ أو ينتهي بشرطة
  • من 3 إلى 63 حرفاً
  • فريد — لا يمكن تكرار نفس الاسم

دومين مخصص

اربط دومينك الخاص بمشروعك على Vox Space مجاناً.

الخطوات

1
افتح إعدادات المشروع

من الداشبورد ← المشروع ← Domains.

2
أضف الدومين

اكتب دومينك واضغط Add. هتحصل على DNS records.

3
حدّث DNS

أضف الـ CNAME record في DNS provider بتاعك.

4
انتظر التفعيل

من دقائق إلى ساعة حسب الـ DNS provider. SSL يُصدر تلقائياً.

DNS Record

DNS Record
Type:  CNAME
Name:  @ (أو www)
Value: cname.voxspace.qzz.io

SSL / HTTPS

كل مشروع على Vox Space محمي تلقائياً بشهادة SSL/TLS بدون أي إعداد.

SSL يُصدر تلقائياً لكل مشروع جديد وعند ربط دومين مخصص. لا يحتاج أي تدخل منك.

التفاصيل

  • شهادة TLS 1.3 على كل المشاريع
  • تجديد تلقائي قبل انتهاء الصلاحية
  • HTTP يُحوّل تلقائياً لـ HTTPS
  • يشمل الدومينات المخصصة

إعدادات المشروع

تحكم في إعدادات مشروعك من الداشبورد.

الإعدادات المتاحة

  • تغيير اسم المشروع والسب-دومين
  • ربط أو فك ربط دومين مخصص
  • عرض سجل عمليات النشر
  • حذف المشروع
  • إعادة النشر من آخر نسخة
